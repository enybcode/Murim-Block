# Jade Qi Sword

Generated starter sword model.

## Files

- Model: `jade_qi_sword.json`
- Texture: `jade_qi_sword.png`

## MCreator Import Targets

- Copy model to `src/main/resources/assets/murim_block/models/custom/jade_qi_sword.json`
- Copy texture to `src/main/resources/assets/murim_block/textures/item/jade_qi_sword.png`
- Item model parent can be `murim_block:custom/jade_qi_sword`
- In MCreator, custom model name should become `jade_qi_sword:default` if imported like existing custom models.

## Before Import

- Open the JSON in Blockbench as Java Block/Item.
- Check Display tab with weapon preset / hand views.
- Paint or replace the placeholder PNG.
- Keep names lowercase with underscores.
- Run `gradlew compileJava` after MCreator import/regeneration.
